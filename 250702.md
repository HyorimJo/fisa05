#### 5. 컨테이너 자료형

1) 리스트(list)
- [] 꺾은괄호, 콤마(,)로 구분
- 원소 간 순서 존재 -> 인덱스 붙여 호출 가능
- 원소 내용 변경 가능
- ex) a = [‘신짱구’, 40000, True]

2) 튜플(tuple)
- () 둥근괄호
- 원소 간 순서 존재 -> 인덱스 붙여 호출 가능
- 원소 내용 변경 불가능
- ex) b = (None, [1,2], 3, 3, 3)
3) 레인지(range)
- 정수 수열생성
- 원소 내용 변경 불가능
